    import ibis

uri = "postgresql+psycopg2://postgres:postgres@db.postgres.app.com/postgres"

    con = ibis.postgres.connect(url=uri)

    Type¶

    Ibis uses its own type aliases that map onto database types. See, for example, the correspondence between Ibis type names and Impala type names:

    Ibis type      Impala Type
~~~~~~~~~      ~~~~~~~~~~~
int8           TINYINT
int16          SMALLINT
int32          INT
int64          BIGINT
float          FLOAT
double         DOUBLE
boolean        BOOLEAN
string         STRING
timestamp      TIMESTAMP
decimal(p, s)  DECIMAL(p,s)
interval(u)    INTERVAL(u)
