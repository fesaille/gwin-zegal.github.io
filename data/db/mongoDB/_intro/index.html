    MongoDB¶

    Notation:

    • field path: $fieldName
    • system variable: $$UPPERCASE
    • user variable: $$foo

    mongod is the main daemon process for mongoDB. Interaction is not performed directly but through a driver. Basic flags for the cli: --port/--dbpath/--logpath/--fork. Server shutdown is performed with db.shutdownServer after selecting the admin db (use admin).
